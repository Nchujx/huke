# 虎课网视频批量下载

### 介绍

研究虎课网视频算法，发现为AES CBC 加密，然后经过一系列复杂的分析，爬取过程，写了此软件，基本支持虎课网所有类型视频下载，

仅为方便用户下载会员到期视频却没时间看的问题，请勿用于非法用途！
